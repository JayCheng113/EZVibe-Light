# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/),
and this project adheres to [Semantic Versioning](https://semver.org/).

## [0.1.8] - 2026-03-14

### Added
- GitHub Actions CI workflow (Node 18/20/22 test matrix)
- GitHub Actions CD workflow (auto-publish to npm on tag push)
- MIT LICENSE file
- CHANGELOG.md with backfilled release history

## [0.1.7] - 2025-06-13

### Changed
- Simplified code structure and improved maintainability

### Added
- 22 edge-case tests for comprehensive coverage

## [0.1.6] - 2025-06-12

### Changed
- Added repository metadata to package.json

## [0.1.5] - 2025-06-11

### Fixed
- Hardened store with improved error handling
- Fixed selection bugs in session and idea lists

### Added
- 21 robustness tests

## [0.1.4] - 2025-06-10

### Added
- Dynamic terminal resize support

## [0.1.3] - 2025-06-09

### Changed
- Clean up session UX — stop instead of kill, hide stopped status

## [0.1.2] - 2025-06-08

### Fixed
- Centralized version from package.json
- Resolved stale closures, scrolling bugs, and removed dead code

### Added
- Session status indicators
- Improved help overlay
- `--no-check` flag to skip tmux/claude prerequisite checks

## [0.1.1] - 2025-06-07

### Fixed
- Resolved stale closures and scrolling bugs
- Cleaned up dead code

## [0.1.0] - 2025-06-06

### Added
- Initial release
- TUI for managing Claude Code sessions with tmux
- Idea management with project association
- Session creation wizard
- Keyboard-driven interface
- Context viewer for Claude Code data
